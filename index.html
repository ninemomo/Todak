Todak Time

개인정보처리방침

시행일 2025년 4월 28일  ·  최종 수정일 2025년 4월 28일

핵심 원칙

토닥은 개발자 서버를 운영하지 않습니다.
모든 데이터는 사용자의 기기 또는 사용자 본인의 iCloud 계정에만 저장됩니다.
개발자는 사용자 데이터에 접근할 수 없습니다.

01

개요

토닥 타임(Todak Time)은 시험을 준비하는 누구나 사용할 수 있는 과목별 집중 타이머 앱입니다.

본 방침은 앱을 사용하는 과정에서 어떤 정보가 어떻게 처리되는지를 명확히 안내합니다.

02

수집하는 정보

토닥이 기기 내에 저장하는 데이터는 다음과 같습니다.

항목내용저장 위치
과목 정보과목명, 색상, 하루 목표 시간기기 / iCloud (선택)
공부 세션 기록시작·종료 시각, 과목명, 공부 시간기기 / iCloud (선택)
시험 일정시험명, 날짜기기 / iCloud (선택)
앱 설정언어, 화면 모드, 햅틱, 알림 간격 등기기 전용

위 정보는 사용자가 직접 입력한 것이며, 앱이 자동으로 생성하거나 외부에서 수집하는 데이터가 아닙니다.

03

수집하지 않는 정보

토닥은 다음 정보를 일절 수집하지 않습니다.

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 위치 정보 (GPS 및 IP 기반 위치 포함)
  • 연락처, 사진, 카메라, 마이크 접근 정보
  • 광고 식별자 (IDFA, IDFV 등)
  • 기기 고유 식별자
  • 검색 기록, 브라우징 기록
  • 타 앱 사용 정보
  • 충돌 리포트 및 자동 진단 데이터
  • 행동 분석 데이터 (애널리틱스 SDK 없음)
  • 소셜 로그인 정보
04

iCloud 동기화

iCloud 동기화를 활성화한 경우, 과목·세션·일정 데이터가 사용자 본인의 iCloud 계정을 통해 기기 간 동기화됩니다.

  • 동기화 데이터는 개발자가 접근할 수 없습니다.
  • Apple의 CloudKit 인프라를 사용하며, Apple 개인정보처리방침이 함께 적용됩니다.
  • iCloud 동기화는 앱 설정에서 언제든지 비활성화할 수 있습니다.
  • 동기화를 비활성화하면 데이터는 해당 기기 내에만 유지됩니다.
05

인앱 결제

프리미엄 기능 구매는 Apple App Store 결제 시스템을 통해서만 처리됩니다.

  • 카드번호, 결제 정보 등은 개발자에게 전달되지 않습니다.
  • 구매 여부 확인은 Apple 서버와 기기 사이에서만 이루어집니다.
  • 환불은 Apple 환불 정책에 따라 처리됩니다.
06

로컬 알림

앱은 사용자가 설정한 경우에 한해 다음 알림을 발송합니다.

  • 타이머 체크인 알림 (설정한 간격마다)
  • 시험 D-day 카운트다운 알림

로컬 알림은 기기 내에서만 생성되며, 외부 서버를 통해 발송되지 않습니다. 알림 권한은 iOS 설정에서 언제든지 철회할 수 있습니다.

07

제3자 제공 및 공유

토닥은 사용자 데이터를 어떤 제3자에게도 판매, 임대, 공유하지 않습니다.

Apple 서비스역할
CloudKitiCloud 기기 간 동기화 (사용자 선택 시에만)
App Store / StoreKit인앱 결제 처리
UNUserNotifications로컬 알림 생성 (서버 없음)

모든 서비스는 Apple 개인정보처리방침이 적용됩니다.

08

데이터 보존 및 삭제

상황처리 결과
앱 삭제기기 내 모든 데이터 즉시 삭제
iCloud 데이터 삭제설정 → [이름] → iCloud → 저장 용량 관리 → 토닥 → 삭제
앱 내 초기화설정 → "전체 데이터 초기화"로 즉시 삭제

개발자는 사용자 데이터를 별도 서버에 보존하지 않으므로, 개발자에게 삭제 요청을 별도로 보낼 필요가 없습니다.

09

아동 개인정보 보호

토닥은 만 14세 미만 아동을 대상으로 개인정보를 의도적으로 수집하지 않습니다. 앱 내에서 개인 식별 정보를 직접 수집하지 않으므로, 연령에 무관하게 동일한 보호 수준이 적용됩니다.

10

사용자의 권리

권리방법
열람앱 내 모든 데이터 직접 확인 가능
수정과목명, 시험 일정 등 앱 내에서 직접 수정
삭제앱 설정 → "전체 데이터 초기화" 또는 앱 삭제
이동(이식)설정 → CSV 내보내기 기능으로 데이터 추출
동기화 중단앱 설정 → iCloud 동기화 비활성화
알림 철회iOS 설정 → 토닥 → 알림 → 비활성화
11

보안

  • 앱 내 데이터는 iOS 기본 샌드박스 보안 체계로 보호됩니다.
  • iCloud 동기화 데이터는 Apple이 암호화하여 전송·저장합니다.
  • 개발자 서버가 없으므로 서버 침해로 인한 데이터 유출 위험이 없습니다.
12

방침 변경

개인정보처리방침이 변경될 경우 본 페이지 상단의 최종 수정일을 업데이트합니다. 중요한 변경사항은 앱 업데이트 노트에 명시합니다. 변경 후에도 앱을 계속 사용하면 변경된 방침에 동의한 것으로 간주합니다.
